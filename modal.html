What does DOM stand for, and what is its purpose in web development?

Answer: DOM stands for Document Object Model. It is a programming interface for web documents that represents the page so that programs can change the document structure, style, and content. The DOM allows developers to manipulate HTML and XML documents using a structured object-oriented model.

What is the method to select an element by its ID in the DOM?

Answer: You can use the document.getElementById() method to select an element by its ID.

How do you change the text content of a DOM element?

You can change the text content of a DOM element by setting the textContent property.

How can you add an event listener to a DOM element?

You can add an event listener using the addEventListener method. For example, to listen for a click event.
